Flex direction

Flex direction: is CSS property which define the direction of items inside the flex container.

Flex direction are mainly two types row wise and column wise.

Flex-direction: row items are in row direction, From left to right. row dirction are default in the flex.

Syntax is: Flex-direction: row ;

Flex-direction: column items are column direction. form top to bottom.

Syntax is: Flex-direction: column

There are also two types row-reverse and column-reverse which define the reverse position of the flex.

gap: which define the gap between two element.

Syntax is: gap:-- px ;

flex-basis: which define the widt for flex row and height for flex column

*Flex column automatically accupy the 100% widt so we need to use different display type.

like: display: inline-flex;
